# biblioteca_digital

É um sistema de gerenciamento de documentos digitais, desenvolvido em python para uma biblioteca digital fictícia

## Sobre o projeto
Este sistema permite que o bibliotecário gerencie documentos digitais como PDF e ePUBs diretamente pelo terminal. reduz o erro do processo manual

## Funcionalidades
1 - lista todos os documentos da biblioteca cadastrados
2 - Inclui novos documentos
3 - Opção para renomear documentos que já estão cadastrados
4 - Excluir um documento

## Como funciona:

## execute:
python main.py 


ao executar o programa python, um menu é exibido com as funcções disponíveis, é só digitar a opção desejada e pressionar enter, para sair do programa digite 5 sair
