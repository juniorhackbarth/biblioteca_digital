# Guia de Contribuição

Obrigado por ter interesse em contribuir com o projeto Biblioteca Digital!

## Como contribuir

### 1. Faça um fork do projeto

Clique no botão Fork no canto superior direito da página do repositório no GitHub.

### 2. Clone o repositório na sua máquina

git clone https://github.com/seu-usuario/biblioteca_digital.git

### 3. Crie uma branch para sua alteração

git checkout -b minha-alteracao

### 4. Faça as alterações no código

Edite os arquivos necessários no VS Code.

### 5. Faça o commit das alterações

git add .
git commit -m "Descrição clara do que foi alterado"

### 6. Envie para o GitHub

git push origin minha-alteracao

### 7. Abra um Pull Request

Acesse o repositório no GitHub, clique em Pull Requests e depois em New Pull Request.
Descreva o que foi alterado e envie para revisão.

## Boas práticas

- Escreva mensagens de commit claras e descritivas
- Mantenha o código organizado e comentado
- Teste as funcionalidades antes de enviar
